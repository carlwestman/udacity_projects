The WWW

  • Collection of HTML docs
  • Connected by Links
  • Can contain many types of files, ie:
    • Plaintext
    • HTML
    • Images
    • Videos
    • Music

    Components of the Web

  • Browser - program for accessing files on web via HTTP protocol
  • Internet - A web of connected servers
  • HTTP - Protocol in which browser commumicates with servers
  • Servers - Computers hosting files served to browsers that request them

    • What is HTML

  • HyperText Markup Language
  • Made up of
    • Text content - What you see
    • Mark up - How it looks
    • Links - References to other files

    HTML Basics

    HTML Tags

  • HTML-tags usually made up of:
    • Opening tag
    • Content
    • Closing tag
  • Tags can contain attributes

    • Bold Tag

    <b>[INSERT CONTENT]</b>
  • Inline Element

    • IMG Tag

    <img src="[insert link reference to img location, URL]" alt="[insert alternative]">
  • Inline Element
  • alt-text will be redered in case src-ref is broken or if browser for other reasons does not render img correctly.
  • alt-text will also be used in case reader is blind and screen reader is used.

    • A tag

    <a href="[Insert link URL]">[Insert Link text]</a>
  • Inline Element

    • P Tag

    <p>[Insert paragraph]</p>
  • Block Element
  • Multi-line by closing and opening new p-tag

    • BR Tag

    [text]<br>[text]
  • Line break

    • SPAN Tag

    <span>[Insert Stuff]</span>
  • Does not do anything except contain stuff
  • Can, through css, change the way things look
  • Inline Element

    • DIV Tag

    <div>[Insert Stuff]</div>
  • Does not do anything except contain stuff
  • Can, through css, change the way things look
  • Block Element

    • Inline vs. Block Elements

  • Inline = just text
  • Block = "invisible box" with height and width

    • Document structure

  • Head = metadata
    • Can contain static files and descriptive elements:
      • Javascript
      • CSS
      • Title
      • Keywords
      • Author
  • Body = Contents
  • Important - makes it readable for others
